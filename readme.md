# 综合课设操作手册

## 文件说明

./my-tiny-compiler: 工程文件夹，运行方式见“工程文件运行环境”部分。

./my_tiny_compiler.exe: 可执行文件，需要在windows或linux的wine环境下运行。

./project.docx: 课程设计报告

./project.pdf: 课程设计报告

./test.txt: 测试输入文件

## 工程文件运行环境

操作系统：windows/linux

编译环境：g++，cmake版本2.8以上

IDE环境：Clion.2022.1

## 可执行文件运行说明

- 运行后输入"test.txt"（文件内容可以修改）
- 根据列出的选项选择需要进行的功能
- 格式化输出文件位于 ./my-tiny-compiler 中，文件名为 out.c
